# Akurati Kaustiki - Portfolio Website

Terminal-style personal portfolio website for a Computer Science Engineer (M.Tech AI/ML).

## Overview

A minimalist, monochrome portfolio with an interactive terminal interface. Built with vanilla HTML, CSS, and JavaScript.

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties for theming)
- Vanilla JavaScript (ES6+ modules)
- JetBrains Mono font

## Features

- Interactive terminal with command execution
- Light/dark theme toggle
- Neural network background pattern
- Responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation

## Terminal Commands

| Command | Description |
|---------|-------------|
| `help()` | List all available commands |
| `df.info` | Display name and title |
| `education.head()` | Show education entries |
| `experience.head()` | Show work experience |
| `skills.head()` | Show technical skills |
| `projects.head()` | Show projects |
| `finearts.head()` | Show fine arts |
| `contact.head()` | Show contact info |
| `clear` | Clear terminal output |
| `theme.toggle()` | Toggle light/dark mode |

## Setup

Open `index.html` in any modern browser. No build step or server required.
