## Workflow

### Phase Execution (One at a time)
Work through phases sequentially. After each phase:
1. Run lint/typecheck
2. Wait for user confirmation: "tested" or "working fine"
3. Only then proceed to next phase

### Before Starting
1. Read `todo.md` - Identify current phase
2. Read `requirements.md` - Understand phase requirements
3. Read `design.md` - Check colors, typography, spacing
4. Read `skill.md` - Follow frontend design guidelines

### Per Phase
1. Implement components/features for current phase
2. Run lint/typecheck and fix any errors
3. Report completion and wait for user testing/approval
